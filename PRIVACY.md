# Privacy Policy for BetterKanban

Last updated: June 15, 2026

## Data Collection

BetterKanban does not collect, transmit, or share any personal data.

## Local Storage

All user settings (column order, visibility, collapse state) are stored locally in the browser using the browser's built-in `storage` API. If browser sync is enabled, settings may be synchronized across devices by the browser itself. BetterKanban does not operate any external servers.

## Permissions

The extension requests permission to run on SharePoint and Microsoft Lists board view URLs so it can read the board layout and apply user preferences. It does not access any other sites or data.

## Contact

For questions, please open an issue at https://github.com/jjspscl/better-kanban/issues.
