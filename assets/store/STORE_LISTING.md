# Store Listing Copy

## Extension Name
BetterKanban

## Short Description (max 132 characters)
Sort, hide, and collapse status columns in SharePoint and Microsoft Lists kanban board views.

## Long Description
BetterKanban gives you control over SharePoint and Microsoft Lists board views.

Features:
- Drag to reorder status columns
- Show or hide columns
- Collapse columns using native SharePoint behavior
- Save layouts per list and per view
- Export and import your settings

All settings are stored locally in your browser. No data is sent to external servers.

Works on SharePoint Online and Microsoft Lists board views.
