# BetterKanban

Browser extension that gives you control over SharePoint and Microsoft Lists kanban board views.

## Features

- Drag to reorder status columns
- Show or hide columns
- Collapse columns using native SharePoint behavior
- Save layouts per list and per view
- Export and import your settings

## Supported sites

- SharePoint Online board views (`*.sharepoint.com/.../Lists/...`)
- Microsoft Lists (`lists.live.com`)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run zip        # Chrome / Edge
npm run zip:firefox # Firefox
```
